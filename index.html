Why learn JavaScript?

Ever wonder what makes JavaScript so interesting? Or why it is in such high demand?

JavaScript can help you:

  • Build websites and web applications
  • Play audio and/or video
  • Validate input forms
  • Animate page elements
  • React to users with events
  • Load new content without refreshing the page
  • Autocomplete input fields

You'd like some more reasons?

  1. JS has a mature ecosystem and community
  2. JS is in very high (and increasing) demand
  3. JS is fun!
